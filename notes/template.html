To use lists in JavaScript, you can use an array. Arrays are a special type of object that can hold multiple values in a single variable. To create an array, you can use the Array constructor or the array literal syntax, which is simpler and easier to read.

Here is an example of how you can create an array using the array literal syntax:

// create an array with three elements
      const myList = ['apple', 'banana', 'orange'];
      
      // access an element in the array using its index
      console.log(myList[0]); // prints "apple"
      
      // add a new element to the end of the array
      myList.push('pear');
      
      // remove an element from the end of the array
      myList.pop();
      
      // find the length of the array
      console.log(myList.length); // prints 4

In JavaScript, arrays are zero-indexed, which means that the first element in the array has an index of 0, the second element has an index of 1, and so on. To access an element in the array, you can use square brackets [] and provide the index of the element you want to access.

You can also add new elements to the end of an array using the push() method, and remove the last element from the array using the pop() method. The length property of an array tells you how many elements the array has.

In addition to the methods and properties mentioned above, there are many other methods and properties available for working with arrays in JavaScript. You can learn more about arrays in JavaScript by reading the documentation or by searching online.
