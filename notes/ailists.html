JavaScript Lists

Lists in JavaScript are used to store multiple values in a single variable. Here are a few ways to create and manipulate lists in JavaScript:

Creating a List

To create a list in JavaScript, you can use the array literal syntax, which is an array of elements enclosed in square brackets:


  var fruits = ["apple", "banana", "cherry"];

Accessing Elements

To access individual elements in a list, you can use square bracket notation with the index of the element you want to access, starting at 0 for the first element:


  // access the first element (index 0)
  var first = fruits[0];
  
  // access the second element (index 1)
  var second = fruits[1];
  
  // access the third element (index 2)
  var third = fruits[2];

Adding and Removing Elements

To add elements to a list, you can use the push() method, which adds elements to the end of the list:


  fruits.push("mango");

To remove elements from a list, you can use the pop() method, which removes the last element from the list:


  fruits.pop();

Iterating Over a List

To iterate over the elements in a list, you can use a for loop:


  for (var i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
  }
